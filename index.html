One-time Firebase setup

  1. Open https://console.firebase.google.com → Add project. Name it anything. Disable Analytics (not needed).
  2. In the project, open Build → Authentication → Sign-in method. Enable Anonymous. Save.
  3. Open Build → Firestore Database → Create database → Start in production mode. Pick any region. Then in the Rules tab paste and publish:
    rules_version='2';service cloud.firestore{match /databases/{d}/documents{match /{p=**}{allow read,write: if request.auth!=null;}}}
  4. Open Project Settings (⚙) → General → Your apps → Web app (</>). Register with any nickname. Copy the firebaseConfig object and paste it into the FIREBASE_CONFIG constant near the top of this file's script, replacing the placeholder values. Save, reload.

Host this HTML somewhere (GitHub Pages, Netlify Drop, Cloudflare Pages, or any static host). iOS Safari will not run this file off a USB / Files app reliably — it needs an https:// URL.
